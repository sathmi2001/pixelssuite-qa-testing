# Playwright Automation Testing

## Overview

This folder contains the Playwright automation testing project developed for the **PixelsSuite Web Application**.

The project demonstrates my practical experience with test automation using Playwright and JavaScript. It includes functional test scripts, browser automation, file upload validation, and UI verification.

---

## Technologies Used

- Playwright
- JavaScript
- Node.js
- Chrome DevTools
